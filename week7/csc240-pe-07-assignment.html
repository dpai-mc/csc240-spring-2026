Detailed Function Specifications

Function 1: factorial(n)

n! — Factorial

Input: Non-negative integer n (0 ≤ n ≤ 20)

Output: n! = n × (n−1) × (n−2) × … × 2 × 1

Edge cases:

  • 0! = 1 (by convention)
  • 1! = 1
  • Negative input → error message
  • n > 20 → warning about overflow (20! is the max for long)

Function 2: permutation(n, r)

P(n, r) — r-Permutation

Input: Non-negative integers n, r where 0 ≤ r ≤ n

Output: P(n, r) = n!/(n−r)!

Edge cases:

  • P(n, 0) = 1 for any n
  • P(n, n) = n!
  • r > n → error message (cannot arrange more items than available)
  • Negative inputs → error message

Implementation tip: Use the product form n × (n−1) × … × (n−r+1) to avoid computing large factorials.

Function 3: combination(n, r)

C(n, r) — r-Combination

Input: Non-negative integers n, r where 0 ≤ r ≤ n

Output: C(n, r) = n!/(r!(n−r)!)

Edge cases:

  • C(n, 0) = C(n, n) = 1
  • r > n → error message
  • Negative inputs → error message

Implementation tip: Use the symmetry property C(n, r) = C(n, n−r) to minimize computation. Use iterative multiplication/division to avoid overflow.

Function 4: permWithRepetition(n, r)

nʳ — Permutation with Repetition

Input: Positive integer n, non-negative integer r

Output: nʳ (n raised to the power r)

Edge cases:

  • n⁰ = 1 for any positive n
  • 0⁰ = 1 (by convention in combinatorics)
  • 0ʳ = 0 for r > 0
  • Negative inputs → error message

Function 5: combWithRepetition(n, r)

C(n+r−1, r) — Combination with Repetition (Stars and Bars)

Input: Positive integer n (types), non-negative integer r (items)

Output: C(n+r−1, r)

Edge cases:

  • r = 0 → 1 (one way to choose nothing)
  • n = 1 → 1 (only one type, must take r of it)
  • Negative inputs → error message

Implementation tip: Reuse your combination() function: combWithRepetition(n, r) = combination(n + r - 1, r)

Function 6: multisetPermutation(n, groups[])

n!/(n₁!×n₂!×…×nₖ!) — Multiset Permutation

Input: Total count n, array of group sizes [n₁, n₂, …, nₖ]

Output: n! / (n₁! × n₂! × … × nₖ!)

Edge cases:

  • Sum of groups must equal n → error if not
  • All groups of size 1 → result is n!
  • Single group of size n → result is 1
  • Empty groups array → error message
