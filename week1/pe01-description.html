üìù Assignment Description

You will create a program that implements basic set operations on sets of integers. Your program should allow users to input two sets and then perform various set operations, displaying the results in a clear, readable format.

What You're Building

A command-line application that:

  • Accepts two sets of integers as input from the user
  • Performs the following set operations:
    • Union (A ‚à™ B): All elements in A or B (or both)
    • Intersection (A ‚à© B): Elements in both A and B
    • Difference (A - B): Elements in A but not in B
    • Difference (B - A): Elements in B but not in A
  • Displays all results in a clear, formatted manner
  • (Optional bonus) Checks if an element is a member of a set

‚úÖ Requirements

Functional Requirements

Your program MUST:

Accept Input:

  • Prompt the user to enter elements for Set A
  • Prompt the user to enter elements for Set B
  • Handle integer input appropriately
  • Accept sets of varying sizes (minimum 3 elements, maximum 10 elements recommended)

Represent Sets:

  • Java: Use HashSet<Integer> or arrays (HashSet recommended for automatic duplicate handling)
  • C++: Use std::set<int> or std::unordered_set<int> or arrays

Implement Operations:

  • Union (A ‚à™ B): Return/display all unique elements from both sets
  • Intersection (A ‚à© B): Return/display only elements present in both sets
  • Difference (A - B): Return/display elements in A that are not in B
  • Difference (B - A): Return/display elements in B that are not in A

Display Output:

  • Show the original Set A
  • Show the original Set B
  • Show the result of each operation clearly labeled
  • Format output for readability (see sample output below)

Technical Requirements

  • Language: Java or C++ (choose ONE)
  • IDE: Visual Studio Code (required for consistency)
  • Input Method: Console/terminal input (Scanner in Java, cin in C++)
  • Output Method: Console/terminal output (System.out in Java, cout in C++)
  • File Structure:
    • Single source file (.java or .cpp)
    • README.txt with compilation and execution instructions

üíª Getting Started

1 Set Up Your Environment

Make sure you've completed the "Programming Environment Setup" assignment:

  • Visual Studio Code installed
  • Java JDK or C++ compiler installed
  • Extensions installed (Java Extension Pack or C/C++ extension)
  • Tested with a "Hello, World!" program

2 Plan Your Approach

Before coding, plan your solution:

Choose your data structure:

  • Java: HashSet<Integer> (recommended) or ArrayList<Integer>
  • C++: std::set<int> or std::unordered_set<int> (recommended) or std::vector<int>

Break down the problem:

  • How will you read input?
  • How will you store sets?
  • How will you implement each operation?
  • How will you display results?

Pseudocode example for Union:

function union(setA, setB):
    result = empty set
    for each element in setA:
        add element to result
    for each element in setB:
        add element to result (duplicates automatically handled)
    return result

3 Start with a Template

Use the starter code templates provided on Canvas:

  • PE01_Template.java (for Java students)
  • pe01_template.cpp (for C++ students)

These templates include:

  • Basic structure and imports
  • Input/output formatting
  • Comments indicating where to add your code

4 Implement One Operation at a Time

  1. Start with Union (easiest)
  2. Then Intersection
  3. Then Difference (A - B)
  4. Finally Difference (B - A)

Test each operation before moving to the next!

5 Test Thoroughly

Test your program with:

  • The sample inputs provided above
  • Edge cases: empty intersection, disjoint sets, identical sets
  • Sets of different sizes
  • Duplicate input (should be handled gracefully)
