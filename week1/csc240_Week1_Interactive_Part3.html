1. Power Sets

Definition: The power set of a set A is the set of all possible subsets of A, including the empty set (‚àÖ) and A itself.

Notation: P(A) or 2^A

üí° Size Formula

If a set A has n elements (denoted |A| = n), then the power set P(A) has 2^n elements.

Check: If A = {1, 2}, |A|=2.
P(A) = { ‚àÖ, {1}, {2}, {1,2} }.
Size is 4, which is 2^2. Correct!

1. From Math to Code

How do we represent a mathematical set in programming?

  • Arrays/Vectors: Simple, but need to handle duplicates manually.
  • Hash Sets (Java HashSet / C++ std::unordered_set): O(1) lookups, automatically handles duplicates.
Code Snippet: Intersection (Java style)
public Set<Integer> intersection(Set<Integer> A, Set<Integer> B) {
    Set<Integer> result = new HashSet<>();
    for (Integer x : A) {
        if (B.contains(x)) {
            result.add(x);
        }
    }
    return result;
}
