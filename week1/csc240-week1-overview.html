CSC 240 – Week 1 Overview

Module Overview

This module introduces the foundational concept of sets, a core building block for the rest of the course. Students learn representation, operations, and visualization of sets.

Required Readings & Media

  • Saylor.org CS202 – Unit 1, Sections 1.1–1.2
  • Interactive Venn Diagram Tool
  • NCTM Set Theory Visualizer

Learning Objectives

  • Define sets using roster and set-builder notation
  • Apply set operations: union, intersection, difference, complement
  • Construct and interpret Venn diagrams
  • Compute power sets and Cartesian products
  • Represent sets programmatically
