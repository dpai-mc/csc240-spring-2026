What is Discrete Mathematics?

Discrete mathematics is the study of mathematical structures that are fundamentally discrete (separate, distinct) rather than continuous.

üí° Example: Discrete vs. Continuous

Discrete Examples:

  • Number of students in class
  • Files on your computer
  • Bits in a byte (0 or 1)
  • Steps in an algorithm

Continuous Examples:

  • Temperature (72.5¬∞F)
  • Distance traveled
  • Time elapsed
  • Volume of water

What is a Set?

A set is a well-defined collection of distinct objects. The objects are called elements or members.

üìù
The Curly Braces { } Symbol:

In mathematics, curly braces { } are the universal symbol for a set. When you see curly braces, you know you're looking at a set. Everything inside the braces is an element of that set.

  • Opening brace { - marks the beginning of a set
  • Elements - listed inside, separated by commas
  • Closing brace } - marks the end of a set

Example: {1, 2, 3} means "the set containing the elements 1, 2, and 3"

üí° Real-World Examples
  • Primary colors: {red, blue, yellow}
    The braces tell us this is a set of three colors
  • Even numbers < 10: {2, 4, 6, 8}
    The braces contain all even numbers less than 10
  • Your programming languages: {Java, Python, C++}
    The braces group these languages into one collection
  • CSC 240 students: {Alice, Bob, Charlie, ...}
    The ... means the set continues with more students

Set Properties:
  1. No order: {1,2,3} = {3,2,1}
  2. No duplicates: {1,2,2,3} = {1,2,3}
  3. Well-defined: Clear membership

Understanding Cardinality

Cardinality is the number of elements in a set. It tells us the "size" of a set.

üí° Cardinality Examples
  • A = {1, 2, 3, 4, 5}
    |A| = 5 because there are 5 elements
  • B = {red, blue}
    |B| = 2 because there are 2 elements
  • C = {Java}
    |C| = 1 because there is 1 element
  • D = {}
    |D| = 0 because there are NO elements (empty set)

The Empty Set (Null Set)

The empty set (also called the null set) is a set that contains no elements. It's like an empty container.

  • { } ‚Üê Empty curly braces (no elements inside)

Set-Builder Notation (Property Method)

Set-builder notation describes properties:

{x | property of x}

Read as: "The set of all x such that x has [property]"

üí° Set-Builder Examples

    All even numbers less than 10
  • {x ‚àà ‚Ñ§ | -5 ‚â§ x < 5} = {-5, -4, -3, -2, -1, 0, 1, 2, 3, 4}
    Integers from -5 to 4 (using the ‚Ñ§ symbol we just learned!)
  • {x | x¬≤ = 9} = {-3, 3}
    All numbers whose square equals 9
  • {x ‚àà ‚Ñï | x < 5} = {1, 2, 3, 4}
    Natural numbers less than 5
  • {n ‚àà ‚Ñ§ | n = 2k for some k ‚àà ‚Ñ§} = {..., -4, -2, 0, 2, 4, ...}
    All even integers (any integer times 2)
