📖 Background: Argument Validity

A logical argument consists of premises (assumptions) and a conclusion. An argument is valid if and only if:

Whenever all premises are true, the conclusion must also be true.

Equivalently, the argument $$P_1, P_2, \ldots, P_n \therefore C$$ is valid if and only if:

$$(P_1 \land P_2 \land \ldots \land P_n) \rightarrow C$$ is a tautology

Example: Modus Ponens

The classic valid argument form:

  • Premise 1: $$P \rightarrow Q$$
  • Premise 2: $$P$$
  • Conclusion: $$Q$$

This is valid because whenever both premises are true, Q must be true.

public class ProofValidator { private ExpressionParser parser; // Reuse from PE-03 private List<String> premises; private String conclusion; public ProofValidator() { this.parser = new ExpressionParser(); this.premises = new ArrayList<>(); } public void addPremise(String premise) { premises.add(premise); } public void setConclusion(String conclusion) { this.conclusion = conclusion; } public ValidationResult validate() { // 1. Extract all variables from premises and conclusion Set<Character> variables = extractVariables(); // 2. Generate all truth assignments List<Map<Character, Boolean>> assignments = generateAssignments(variables); // 3. For each assignment, evaluate premises and conclusion List<Row> criticalRows = new ArrayList<>(); for (Map<Character, Boolean> assignment : assignments) { boolean allPremisesTrue = evaluateAllPremises(assignment); boolean conclusionTrue = evaluateConclusion(assignment); if (allPremisesTrue) { criticalRows.add(new Row(assignment, conclusionTrue)); } } // 4. Check if conclusion is true in ALL critical rows boolean isValid = criticalRows.stream() .allMatch(row -> row.conclusionTrue); return new ValidationResult(isValid, criticalRows); } // Additional helper methods... }

C++ Implementation Skeleton

class ProofValidator { private: ExpressionParser parser; // Reuse from PE-03 vector<string> premises; string conclusion; public: void addPremise(const string& premise) { premises.push_back(premise); } void setConclusion(const string& conc) { conclusion = conc; } ValidationResult validate() { // 1. Extract all variables set<char> variables = extractVariables(); // 2. Generate all truth assignments vector<map<char, bool>> assignments = generateAssignments(variables); // 3. Find critical rows and check validity vector<CriticalRow> criticalRows; bool isValid = true; for (const auto& assignment : assignments) { bool allPremisesTrue = evaluateAllPremises(assignment); bool conclusionTrue = evaluateConclusion(assignment); if (allPremisesTrue) { criticalRows.push_back({assignment, conclusionTrue}); if (!conclusionTrue) { isValid = false; } } } return ValidationResult{isValid, criticalRows}; } };

ARGUMENT IS VALID [Truth table showing validity]

  • Debug with print statements: Output intermediate values to trace logic

⚠️ Common Pitfalls

  • Operator precedence: Ensure your parser handles precedence correctly
  • Variable extraction: Don't miss variables that appear only in conclusion
  • Empty critical rows: If premises are never all true, argument is vacuously valid
  • Output formatting: Align columns properly for readability
