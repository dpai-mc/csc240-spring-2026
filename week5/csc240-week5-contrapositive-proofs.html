📚 Part A: Proof by Contrapositive

🔑 Key Concept: The Contrapositive

To prove $$P \rightarrow Q$$ by contrapositive, we instead prove $$\neg Q \rightarrow \neg P$$.

These statements are logically equivalent, so proving one proves the other.

$$(P \rightarrow Q) \equiv (\neg Q \rightarrow \neg P)$$

When to use: When assuming $$\neg Q$$ gives you something concrete to work with, or when a direct proof seems difficult.

📖 Example: Proof by Contrapositive

Theorem: For any integer $$n$$, if $$n^2$$ is even, then $$n$$ is even.

Contrapositive: If $$n$$ is odd, then $$n^2$$ is odd.

Proof (by contrapositive):

We prove the contrapositive: if $$n$$ is odd, then $$n^2$$ is odd.

Assume $$n$$ is odd. (Premise of contrapositive)

By definition, $$n = 2k + 1$$ for some integer $$k$$.

Then $$n^2 = (2k + 1)^2 = 4k^2 + 4k + 1 = 2(2k^2 + 2k) + 1$$.

Let $$m = 2k^2 + 2k$$. Since $$k$$ is an integer, $$m$$ is an integer.

Therefore, $$n^2 = 2m + 1$$, which is odd by definition.

Since the contrapositive is true, the original statement is true.

∎

📚 Part B: Disproof by Counterexample

🔑 Key Concept: Counterexamples

To disprove a universal statement "For all $$x$$, $$P(x)$$" we only need to find one value where $$P(x)$$ is false.

To disprove $$\forall x, P(x)$$: Find one $$x_0$$ such that $$P(x_0)$$ is false.

Important: A counterexample must be specific and you must verify that it actually makes the statement false.

📖 Example: Disproof by Counterexample

False Claim: For all integers $$n$$, $$n^2 > n$$.

Disproof:

We provide a counterexample to show this statement is false.

Let $$n = 1$$.

Then $$n^2 = 1^2 = 1$$ and $$n = 1$$.

We have $$n^2 = 1 \not> 1 = n$$.

Since $$n^2 > n$$ is false when $$n = 1$$, the statement is disproved.

∎

📊 Comparison: When to Use Each Technique

Technique Use When Goal
Direct Proof The path from $$P$$ to $$Q$$ is clear Prove $$P \rightarrow Q$$
Contrapositive $$\neg Q$$ gives concrete info to work with Prove $$\neg Q \rightarrow \neg P$$
Counterexample You believe the statement is FALSE Find one case where statement fails

⚠️ Common Mistakes to Avoid

  • Contrapositive errors: Make sure you negate BOTH parts correctly (and flip the direction)
  • Incomplete counterexamples: Always VERIFY your counterexample actually fails
  • Using examples as proofs: One example doesn't prove a universal statement true
  • Wrong technique: Don't try to disprove something that's actually true!
