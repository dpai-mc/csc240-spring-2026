In Week 4, you learned that (P → Q) ≡ (¬Q → ¬P). This logical equivalence is the foundation of proof by contrapositive. Now you'll see how to use this equivalence as a powerful proof technique!

1. The Contrapositive Equivalence: Review and Proof

📖 Definition: Contrapositive

The contrapositive of a conditional statement P → Q is the statement ¬Q → ¬P.

In words: The contrapositive of "If P, then Q" is "If not Q, then not P."

Visualizing the Transformation

From Original to Contrapositive

Original Statement
P → Q
"If P, then Q"
⟺
Contrapositive
¬Q → ¬P
"If not Q, then not P"

Key Point: These two statements are logically equivalent—they have the same truth value in all cases!

Why Are They Equivalent? The Truth Table Proof

Let's verify the equivalence using a truth table:

P Q ¬P ¬Q P → Q ¬Q → ¬P
T T F F T T
T F F T F F
F T T F T T
F F T T T T

📐 Theorem: Contrapositive Equivalence

For any statements P and Q:

(P → Q) ≡ (¬Q → ¬P)

The last two columns of the truth table are identical, proving the equivalence.

The Four Related Statements: A Complete Picture

Given a conditional P → Q, here are all four related statements:

Name Form Equivalent To
Original P → Q Contrapositive
Converse Q → P Inverse
Inverse ¬P → ¬Q Converse
Contrapositive ¬Q → ¬P Original

⚠️ Critical Reminder

Original ≡ Contrapositive (always have the same truth value)

Converse ≡ Inverse (always have the same truth value)

Original ≢ Converse (may have different truth values!)

Don't confuse contrapositive with converse—they are very different!

2. What is Proof by Contrapositive?

📖 Definition: Proof by Contrapositive

A proof by contrapositive is a method of proving a conditional statement P → Q by instead proving its contrapositive ¬Q → ¬P.

Since the original and contrapositive are logically equivalent, proving one proves the other.

The Contrapositive Proof Template

📋 Proof by Contrapositive Template

  1. State the theorem: "Theorem: If P, then Q."
  2. Announce the method: "Proof (by contrapositive):" or "We prove the contrapositive."
  3. State the contrapositive: "We will show: If ¬Q, then ¬P."
  4. Assume ¬Q: "Assume ¬Q." (Assume the negation of the conclusion)
  5. Derive ¬P: Use definitions and logical steps to show ¬P follows.
  6. Conclude: "Therefore, ¬Q → ¬P, which proves P → Q by contrapositive."
  7. End the proof: ∎

Comparing Direct Proof and Contrapositive Proof

Direct Proof vs. Contrapositive Proof

Direct Proof of P → Q
  1. Assume P is true
  2. Apply definitions to P
  3. Use logical/algebraic steps
  4. Derive Q
  5. Conclude P → Q
Contrapositive Proof of P → Q
  1. Assume ¬Q is true
  2. Apply definitions to ¬Q
  3. Use logical/algebraic steps
  4. Derive ¬P
  5. Conclude P → Q

💡 Key Insight: Same Destination, Different Starting Point

Both methods prove the same statement P → Q. The difference is:

  • Direct: Start with P, end with Q
  • Contrapositive: Start with ¬Q, end with ¬P

Sometimes starting with ¬Q gives you more useful information to work with!

3. When to Use Proof by Contrapositive

Proof by contrapositive is especially useful in certain situations. Here's a guide to help you decide:

🤔 Should I Use Contrapositive?

1. Does the conclusion Q involve a negative condition or "not"?
→ Yes: Contrapositive might help! ¬Q becomes a positive statement.
→ No: Continue to next question.
2. Does assuming ¬Q give you more concrete information than assuming P?
→ Yes: Use contrapositive! (Classic example: "n is even" gives n = 2k)
→ No: Continue to next question.
3. Have you tried direct proof and gotten stuck?
→ Yes: Try contrapositive as an alternative approach.
→ No: Direct proof is probably fine.

Common Scenarios Where Contrapositive Excels

Scenario Why Contrapositive Helps Example
Proving "If A, then B is [parity]" ¬B gives a concrete form (n = 2k or n = 2k+1) "If n² is even, then n is even"
Conclusion involves "not divisible" ¬Q becomes "divisible," which is easier to use "If 3∤n², then 3∤n"
Hypothesis is abstract, conclusion is concrete Starting with ¬Q (concrete) is easier "If ab is irrational, then a or b is irrational"
Statement has form "If not A, then not B" Contrapositive removes double negatives "If x ≠ y, then f(x) ≠ f(y)"

📝 The Classic Example: Why Contrapositive is Better

Theorem: If n² is even, then n is even.

Direct Proof Attempt:

  • Assume n² is even
  • Then n² = 2k for some integer k
  • So n = √(2k)... but this doesn't obviously give us n = 2m!
  • We're stuck! 😕

Contrapositive Proof:

  • Prove: If n is odd, then n² is odd
  • Assume n is odd
  • Then n = 2k + 1 for some integer k
  • So n² = (2k + 1)² = 4k² + 4k + 1 = 2(2k² + 2k) + 1
  • This is odd! ✓

Lesson: Starting with "n is odd" gives us a concrete form (n = 2k + 1) that we can square. Starting with "n² is even" only gives us n² = 2m, which is hard to "unsquare."

4. How to Form the Contrapositive Correctly

Forming the contrapositive requires careful attention to negation. Here's a systematic approach:

📋 Steps to Form the Contrapositive

  1. Identify P and Q in the statement "If P, then Q"
  2. Form ¬Q (negate the conclusion)
  3. Form ¬P (negate the hypothesis)
  4. Write the contrapositive: "If ¬Q, then ¬P"

Negation Rules You'll Need

Original Statement Negation Rule
n is even n is odd Integers are either even or odd
n is odd n is even Integers are either even or odd
a | b (a divides b) a ∤ b (a does not divide b) Direct negation
x = y x ≠ y Direct negation
x < y x ≥ y Negation of strict inequality
P ∧ Q (P and Q) ¬P ∨ ¬Q (not P or not Q) De Morgan's Law
P ∨ Q (P or Q) ¬P ∧ ¬Q (not P and not Q) De Morgan's Law

📝 Example: Forming Contrapositives

Example 1: "If n is divisible by 6, then n is divisible by 3."

  • P: n is divisible by 6
  • Q: n is divisible by 3
  • ¬Q: n is NOT divisible by 3
  • ¬P: n is NOT divisible by 6
  • Contrapositive: "If n is not divisible by 3, then n is not divisible by 6."

Example 2: "If x = 0 and y = 0, then xy = 0."

  • P: x = 0 AND y = 0
  • Q: xy = 0
  • ¬Q: xy ≠ 0
  • ¬P: ¬(x = 0 AND y = 0) = (x ≠ 0 OR y ≠ 0) (De Morgan's Law)
  • Contrapositive: "If xy ≠ 0, then x ≠ 0 or y ≠ 0."

Example 3: "If n² is even, then n is even."

  • P: n² is even
  • Q: n is even
  • ¬Q: n is odd
  • ¬P: n² is odd
  • Contrapositive: "If n is odd, then n² is odd."

⚠️ Common Mistakes When Forming Contrapositives

Mistake 1: Confusing contrapositive with converse

  • Original: If P, then Q
  • Contrapositive: If ¬Q, then ¬P ✓
  • Converse: If Q, then P ✗ (This is NOT the contrapositive!)

Mistake 2: Forgetting De Morgan's Laws

  • Original: "If A and B, then C"
  • Wrong contrapositive: "If ¬C, then ¬A and ¬B" ✗
  • Correct contrapositive: "If ¬C, then ¬A or ¬B" ✓

Mistake 3: Incorrectly negating inequalities

  • ¬(x < 5) = x ≥ 5, NOT x > 5
  • ¬(x ≤ 5) = x > 5, NOT x ≥ 5

5. Complete Contrapositive Proofs

Let's work through several complete contrapositive proofs, from classic examples to more challenging ones.

📝 Example 1: The Classic - If n² is even, then n is even

Theorem: For any integer n, if n² is even, then n is even.

Theorem: For any integer n, if n² is even, then n is even.

Proof (by contrapositive):

We prove the contrapositive: If n is odd, then n² is odd.

Assume n is an odd integer. (Assumption of ¬Q)

By definition of odd, n = 2k + 1 for some integer k. (Definition)

Then n² = (2k + 1)² (Substitution)

= 4k² + 4k + 1 (Expanding)

= 2(2k² + 2k) + 1. (Factoring)

Let m = 2k² + 2k. Since k is an integer, m is an integer. (Closure)

Thus n² = 2m + 1 where m is an integer. (Substitution)

By definition of odd, n² is odd. (Definition)

This proves the contrapositive, therefore the original statement holds.

∎

📝 Example 2: If n³ is odd, then n is odd

Theorem: For any integer n, if n³ is odd, then n is odd.

Theorem: For any integer n, if n³ is odd, then n is odd.

Proof (by contrapositive):

We prove the contrapositive: If n is even, then n³ is even.

Assume n is an even integer. (Assumption of ¬Q)

By definition of even, n = 2k for some integer k. (Definition)

Then n³ = (2k)³ = 8k³ = 2(4k³). (Algebra)

Let m = 4k³. Since k is an integer, m is an integer. (Closure)

Thus n³ = 2m where m is an integer. (Substitution)

By definition of even, n³ is even. (Definition)

This proves the contrapositive, therefore the original statement holds.

∎

📝 Example 3: Divisibility - If 3∤n², then 3∤n

Theorem: For any integer n, if 3 does not divide n², then 3 does not divide n.

(Notation: 3∤n means "3 does not divide n")

Theorem: For any integer n, if 3∤n², then 3∤n.

Proof (by contrapositive):

We prove the contrapositive: If 3 | n, then 3 | n².

Assume 3 | n. (Assumption of ¬Q)

By definition of divisibility, n = 3k for some integer k. (Definition)

Then n² = (3k)² = 9k² = 3(3k²). (Algebra)

Let m = 3k². Since k is an integer, m is an integer. (Closure)

Thus n² = 3m where m is an integer. (Substitution)

By definition of divisibility, 3 | n². (Definition)

This proves the contrapositive, therefore the original statement holds.

∎

📝 Example 4: If 5∤n², then 5∤n

Theorem: For any integer n, if 5 does not divide n², then 5 does not divide n.

Theorem: For any integer n, if 5∤n², then 5∤n.

Proof (by contrapositive):

We prove the contrapositive: If 5 | n, then 5 | n².

Assume 5 | n. (Assumption)

By definition, n = 5k for some integer k. (Definition)

Then n² = (5k)² = 25k² = 5(5k²). (Algebra)

Let m = 5k². Since k is an integer, m is an integer. (Closure)

Thus n² = 5m where m is an integer, so 5 | n². (Definition)

∎

📝 Example 5: If x + y is irrational, then x is irrational or y is irrational

Theorem: For real numbers x and y, if x + y is irrational, then x is irrational or y is irrational.

First, let's form the contrapositive:

  • P: x + y is irrational
  • Q: x is irrational OR y is irrational
  • ¬Q: x is rational AND y is rational (De Morgan's Law)
  • ¬P: x + y is rational
  • Contrapositive: If x is rational and y is rational, then x + y is rational.

Theorem: If x + y is irrational, then x is irrational or y is irrational.

Proof (by contrapositive):

We prove the contrapositive: If x is rational and y is rational, then x + y is rational.

Assume x and y are both rational. (Assumption)

By definition, x = a/b for integers a, b with b ≠ 0. (Definition)

By definition, y = c/d for integers c, d with d ≠ 0. (Definition)

Then x + y = a/b + c/d = (ad + bc)/(bd). (Algebra)

Let p = ad + bc and q = bd. (Substitution)

Since a, b, c, d are integers, p and q are integers. (Closure)

Since b ≠ 0 and d ≠ 0, we have q = bd ≠ 0. (Nonzero product)

Thus x + y = p/q where p, q are integers with q ≠ 0. (Substitution)

By definition, x + y is rational. (Definition)

This proves the contrapositive, therefore the original statement holds.

∎

📝 Example 6: If n² is not divisible by 4, then n is odd

Theorem: For any integer n, if n² is not divisible by 4, then n is odd.

Forming the contrapositive:

  • P: 4 ∤ n²
  • Q: n is odd
  • ¬Q: n is even
  • ¬P: 4 | n²
  • Contrapositive: If n is even, then 4 | n².

Theorem: If 4 ∤ n², then n is odd.

Proof (by contrapositive):

We prove the contrapositive: If n is even, then 4 | n².

Assume n is even. (Assumption)

By definition, n = 2k for some integer k. (Definition)

Then n² = (2k)² = 4k². (Algebra)

Since n² = 4k² = 4(k²) and k² is an integer, we have 4 | n². (Definition)

∎

6. Side-by-Side Comparison: Direct vs. Contrapositive

Sometimes the same theorem can be proved both ways. Let's compare both approaches for the same statement.

📝 Theorem: If n is odd, then n² is odd

This theorem can be proved directly (and we did in Module 5.2). Let's see both proofs side by side.

Two Proofs of the Same Theorem

Direct Proof

Prove: If n is odd, then n² is odd.

Assume n is odd.

Then n = 2k + 1.

n² = (2k+1)² = 4k² + 4k + 1

= 2(2k² + 2k) + 1

This is odd. ∎

Contrapositive Proof

Prove: If n² is even, then n is even.

Assume n² is even.

Then n² = 2m... 🤔

Hard to proceed!

Direct proof is easier here.

Conclusion: For "If n is odd, then n² is odd," direct proof is easier!

📝 Theorem: If n² is even, then n is even

Now let's compare both approaches for the converse statement.

Two Proofs of the Same Theorem

Direct Proof (Attempt)

Prove: If n² is even, then n is even.

Assume n² is even.

Then n² = 2k.

So n = √(2k)... 🤔

Stuck! Can't show n = 2m.

Contrapositive Proof

Prove: If n is odd, then n² is odd.

Assume n is odd.

Then n = 2k + 1.

n² = 4k² + 4k + 1 = 2(2k²+2k) + 1

This is odd. ∎

Conclusion: For "If n² is even, then n is even," contrapositive is much easier!

💡 Pattern Recognition

Notice the pattern:

  • "If [property of n], then [property of n²]" → Direct proof is usually easier (you can compute n²)
  • "If [property of n²], then [property of n]" → Contrapositive is usually easier (you can't easily "unsquare")

The key insight: It's easier to compute n² from n than to deduce properties of n from n².

10. Summary and Key Takeaways

📋 Key Takeaways from Module 5.4

  1. Contrapositive equivalence: (P → Q) ≡ (¬Q → ¬P)
  2. Proof by contrapositive: To prove P → Q, prove ¬Q → ¬P instead.
  3. When to use it: When assuming ¬Q gives more useful information than assuming P, especially for statements like "If [property of n²], then [property of n]."
  4. Forming the contrapositive: Negate both P and Q, then swap their positions. Remember De Morgan's Laws!
  5. Don't confuse with converse: Contrapositive (¬Q → ¬P) ≠ Converse (Q → P).
  6. Both methods are valid: Direct and contrapositive proofs are equally valid; choose based on which is easier.
