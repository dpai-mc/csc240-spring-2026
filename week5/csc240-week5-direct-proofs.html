📖 Example: Model Direct Proof

Example Proof Structure

Theorem: If n is an even integer, then n² is an even integer.

Proof:

Let n be an even integer. (Premise)

By definition of even, there exists an integer k such that n = 2k. (Definition of even)

Then n² = (2k)² = 4k² = 2(2k²). (Algebra)

Let m = 2k². Since k is an integer, m is also an integer. (Closure of integers under multiplication)

Therefore, n² = 2m where m is an integer.

By definition of even, n² is even.

∎ (QED)

📝 Key Elements to Notice:

  • Clear statement of what we're assuming (premise)
  • Explicit definitions cited when used
  • Step-by-step algebra showing the transformation
  • Justification for why intermediate values are integers
  • Conclusion ties back to the definition

⚠️ Common Mistakes to Avoid

  • Assuming what you're trying to prove - Don't use the conclusion as a premise
  • Skipping steps - Include all algebraic manipulations
  • Not citing definitions - Always state the definition you're using
  • Using specific examples - Proofs must work for ALL cases, not just examples
  • Forgetting to conclude - Explicitly state that you've proven the theorem

💡 Tips for Success

  • Start with definitions: Write out the formal definition of each term in the theorem
  • Work forward from premises: What can you derive from your assumptions?
  • Keep the goal in mind: What form does your conclusion need to take?
  • Justify every step: Ask yourself "why is this step valid?"
  • Read your proof aloud: Does it make logical sense when spoken?
