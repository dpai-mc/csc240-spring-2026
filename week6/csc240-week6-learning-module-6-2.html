🔢 Classic Example 1: The Irrationality of √2

The proof that √2 is irrational is perhaps the most famous proof by contradiction in all of mathematics. It dates back to ancient Greece and is attributed to the Pythagoreans (around 500 BCE).

📜 Historical Note

According to legend, the discovery that √2 is irrational was so disturbing to the Pythagoreans—who believed all numbers could be expressed as ratios of integers—that they allegedly drowned the mathematician Hippasus for revealing this truth. While the story is likely apocryphal, it illustrates how revolutionary this discovery was.

Background: Rational Numbers and Lowest Terms

📖 Definition: Rational Number

A real number r is rational if it can be expressed as r = a/b where a and b are integers with b ≠ 0.

📖 Definition: Lowest Terms

A fraction a/b is in lowest terms (or reduced form) if gcd(a, b) = 1, meaning a and b share no common factors other than 1.

💡 Key Fact

Every rational number can be expressed as a fraction in lowest terms. If a/b is not in lowest terms, we can divide both a and b by their greatest common divisor to reduce it.

The Theorem and Its Proof

📐 Theorem

√2 is irrational.

Let's work through this proof step by step, with detailed explanations at each stage.

Assume, for the sake of contradiction, that √2 is rational.

This means √2 can be written as a fraction of two integers.

Write √2 = a/b where a, b ∈ ℤ, b ≠ 0, and gcd(a, b) = 1.

We can always reduce a fraction to lowest terms, so this assumption is valid. The condition gcd(a, b) = 1 means a and b share no common factors.

From √2 = a/b, we get 2 = a²/b², so 2b² = a².

This algebraic manipulation is straightforward: square both sides, then multiply by b².

Since 2b² = a², we have a² = 2b², which means a² is even.

Any number equal to 2 times something is even by definition.

Since a² is even, a must be even.

This uses the fact that the square of an odd number is odd. (If a were odd, a² would be odd.) Therefore a = 2k for some integer k.

Substituting a = 2k into 2b² = a²:

2b² = (2k)² = 4k²
b² = 2k²

Dividing both sides by 2 gives us b² = 2k².

Since b² = 2k², we have b² is even, so b must be even.

The same reasoning as Step 5 applies here.

Both a and b are even, so they share a common factor of 2.

But we assumed gcd(a, b) = 1, meaning they share NO common factors!

This is a contradiction!

Our assumption that √2 is rational must be false. Therefore, √2 is irrational. ∎

The Complete Proof (Written Form)

Proof that √2 is irrational

Suppose, for the sake of contradiction, that √2 is rational. Then √2 = a/b for some integers a and b with b ≠ 0 and gcd(a, b) = 1 (the fraction is in lowest terms).

Squaring both sides: 2 = a²/b², so 2b² = a².

This means a² is even, so a is even. Write a = 2k for some integer k.

Substituting: 2b² = (2k)² = 4k², so b² = 2k².

This means b² is even, so b is even.

Since both a and b are even, they share a common factor of 2, contradicting gcd(a, b) = 1.

Therefore, √2 is irrational. ∎

Key Lemma: Even Squares

The proof relies on a crucial fact that we should verify:

📐 Lemma

For any integer n: n² is even if and only if n is even.

Proof of the Lemma ▼

(→) If n is even, then n² is even:

If n is even, then n = 2k for some integer k. Thus n² = (2k)² = 4k² = 2(2k²), which is even.

(←) If n² is even, then n is even:

We prove the contrapositive: if n is odd, then n² is odd.

If n is odd, then n = 2k + 1 for some integer k. Thus:

n² = (2k + 1)² = 4k² + 4k + 1 = 2(2k² + 2k) + 1

This is of the form 2m + 1, so n² is odd. ∎

🔢 Extending to Other Square Roots

The technique used for √2 can be extended to prove the irrationality of other square roots. Let's see how the pattern generalizes.

The Irrationality of √3

📐 Theorem

√3 is irrational.

The proof follows the same structure, but we need a different lemma about divisibility by 3.

📐 Lemma

For any integer n: if 3 | n² (3 divides n²), then 3 | n.

Proof of the Lemma (using contrapositive) ▼

We prove the contrapositive: if 3 ∤ n, then 3 ∤ n².

If 3 does not divide n, then n ≡ 1 (mod 3) or n ≡ 2 (mod 3).

Case 1: If n ≡ 1 (mod 3), then n² ≡ 1 (mod 3), so 3 ∤ n².

Case 2: If n ≡ 2 (mod 3), then n² ≡ 4 ≡ 1 (mod 3), so 3 ∤ n².

In both cases, 3 ∤ n². ∎

Proof that √3 is irrational

Suppose, for contradiction, that √3 is rational. Then √3 = a/b where a, b ∈ ℤ, b ≠ 0, and gcd(a, b) = 1.

Squaring: 3 = a²/b², so 3b² = a².

Thus 3 | a², which by our lemma means 3 | a. Write a = 3k.

Substituting: 3b² = 9k², so b² = 3k².

Thus 3 | b², which means 3 | b.

Both a and b are divisible by 3, contradicting gcd(a, b) = 1.

Therefore, √3 is irrational. ∎

General Pattern: √p for Prime p

📐 Theorem

For any prime number p, √p is irrational.

💡 The Key Insight

The proof works because of a special property of primes: if p | ab, then p | a or p | b. This is called Euclid's Lemma. In particular, if p | n², then p | n · n, so p | n.

Proof that √p is irrational for any prime p ▼

Let p be prime. Suppose, for contradiction, that √p is rational.

Then √p = a/b where gcd(a, b) = 1.

Squaring: p = a²/b², so pb² = a².

Thus p | a². Since p is prime, p | a (by Euclid's Lemma).

Write a = pk. Then pb² = p²k², so b² = pk².

Thus p | b², so p | b.

Both a and b are divisible by p, contradicting gcd(a, b) = 1.

Therefore, √p is irrational. ∎

What About Non-Primes?

⚠️ Important Distinction

Not all square roots are irrational! For example:

  • √4 = 2 (rational)
  • √9 = 3 (rational)
  • √16 = 4 (rational)

√n is rational if and only if n is a perfect square.

∞ Classic Example 2: The Infinitude of Primes

Our second classic proof demonstrates that there are infinitely many prime numbers. This proof, attributed to Euclid (circa 300 BCE), is considered one of the most elegant proofs in all of mathematics.

📜 Historical Note

This proof appears in Euclid's Elements (Book IX, Proposition 20). It has been called "the most beautiful proof in mathematics" and remains unchanged after more than 2,300 years.

Background: Prime Numbers

📖 Definition: Prime Number

An integer p > 1 is prime if its only positive divisors are 1 and p itself.

An integer n > 1 that is not prime is called composite.

💡 Fundamental Theorem of Arithmetic

Every integer greater than 1 can be expressed as a product of primes in exactly one way (up to ordering). In particular, every integer greater than 1 has at least one prime divisor.

The Theorem and Its Proof

📐 Theorem (Euclid)

There are infinitely many prime numbers.

Suppose, for the sake of contradiction, that there are only finitely many primes.

Let's call them p₁, p₂, p₃, ..., pₙ. This is the complete list of all primes.

Consider the number N = p₁ · p₂ · p₃ · ... · pₙ + 1

N is the product of all primes, plus 1. This is a specific, well-defined integer greater than 1.

By the Fundamental Theorem of Arithmetic, N has at least one prime divisor.

Call this prime divisor q. Then q | N.

Since our list contains ALL primes, q must be one of p₁, p₂, ..., pₙ.

Let's say q = pᵢ for some i.

Since q = pᵢ, we know q divides the product p₁ · p₂ · ... · pₙ.

So q | (p₁ · p₂ · ... · pₙ).

Since q | N and q | (p₁ · p₂ · ... · pₙ), we have:

q | (N − p₁ · p₂ · ... · pₙ) = 1

If a number divides two quantities, it divides their difference.

So q | 1. But q is prime, so q ≥ 2.

No integer ≥ 2 can divide 1. This is a contradiction!

Our assumption that there are finitely many primes must be false.

Therefore, there are infinitely many primes. ∎

The Complete Proof (Written Form)

Proof of the Infinitude of Primes

Suppose, for the sake of contradiction, that there are only finitely many primes: p₁, p₂, ..., pₙ.

Consider N = p₁ · p₂ · ... · pₙ + 1.

Since N > 1, N has a prime divisor q. Since our list contains all primes, q = pᵢ for some i.

Now, q | N and q | (p₁ · p₂ · ... · pₙ), so q | (N − p₁ · p₂ · ... · pₙ) = 1.

But q ≥ 2, so q cannot divide 1. Contradiction!

Therefore, there are infinitely many primes. ∎

Common Misconceptions

⚠️ What the Proof Does NOT Say

A common mistake is thinking the proof shows that N itself is prime. This is NOT what we prove!

  • N might be prime (like 2 · 3 + 1 = 7)
  • N might be composite (like 2 · 3 · 5 · 7 · 11 · 13 + 1 = 30031 = 59 × 509)

The proof shows that N has a prime factor NOT in our original list—it doesn't claim N is prime.

📝 Additional Contradiction Proofs

Let's examine a few more examples that demonstrate the versatility of proof by contradiction.

Example: No Integer Solutions

📐 Theorem

There are no integers a and b such that 18a + 6b = 1.

View Proof ▼

Proof

Suppose, for contradiction, that there exist integers a and b with 18a + 6b = 1.

Factor the left side: 6(3a + b) = 1.

This means 6 | 1 (6 divides 1).

But 6 > 1, so 6 cannot divide 1. Contradiction!

Therefore, no such integers exist. ∎

Example: Uniqueness of Additive Identity

📐 Theorem

Zero is the unique additive identity for real numbers.

View Proof ▼

Proof

Existence: 0 is an additive identity since x + 0 = x for all real x. ✓

Uniqueness (by contradiction):

Suppose there are two additive identities, 0 and z, where z ≠ 0.

Since 0 is an identity: z + 0 = z

Since z is an identity: z + 0 = 0

Therefore z = 0, contradicting z ≠ 0.

So 0 is the unique additive identity. ∎

Example: Between Any Two Rationals

📐 Theorem

Between any two distinct rational numbers, there exists another rational number.

View Proof (Direct, but instructive) ▼

Proof (Constructive)

Let r and s be rational numbers with r < s.

Consider m = (r + s)/2, the midpoint.

Since r and s are rational, r + s is rational, and (r + s)/2 is rational.

Also, r < (r + s)/2 < s (the midpoint lies strictly between).

Therefore, m is a rational number between r and s. ∎

Note: This is a direct/constructive proof. We could also prove it by contradiction: assume no rational exists between r and s, then derive a contradiction.

Computer Science Application: Impossibility Results

Preview: The Halting Problem

One of the most important results in computer science—that no program can determine whether an arbitrary program halts—is proved by contradiction.

Sketch of the argument:

  1. Assume a program H(P, I) exists that returns TRUE if program P halts on input I, and FALSE otherwise.
  2. Construct a new program D that:
    • Takes a program P as input
    • Runs H(P, P)
    • If H returns TRUE, D loops forever
    • If H returns FALSE, D halts
  3. Ask: Does D(D) halt?
  4. If D(D) halts, then H(D, D) = FALSE, so D should loop. Contradiction!
  5. If D(D) loops, then H(D, D) = TRUE, so D should halt. Contradiction!
  6. Therefore, H cannot exist.

You'll study this in more detail in Theory of Computation courses.

Exploring √2 Approximations

/**
 * Demonstrates that √2 cannot be exactly represented as a ratio of integers.
 * We search for a/b where (a/b)² = 2, but never find an exact match.
 */
public class IrrationalityExplorer {
    
    public static void main(String[] args) {
        exploreSquareRootTwo(1000);
        verifyLowestTermsContradiction();
    }
    
    /**
     * Searches for rational approximations to √2.
     * Shows that we get close but never exact.
     */
    public static void exploreSquareRootTwo(int maxDenom) {
        System.out.println("Searching for a/b where (a/b)² = 2...");
        System.out.println("Best approximations found:\n");
        
        double sqrt2 = Math.sqrt(2);
        double bestError = Double.MAX_VALUE;
        
        for (int b = 1; b <= maxDenom; b++) {
            // Find the a that makes a/b closest to √2
            int a = (int) Math.round(b * sqrt2);
            
            double ratio = (double) a / b;
            double squared = ratio * ratio;
            double error = Math.abs(squared - 2.0);
            
            if (error < bestError) {
                bestError = error;
                System.out.printf("  %d/%d = %.10f, squared = %.10f, error = %.2e%n",
                    a, b, ratio, squared, error);
            }
            
            // Check if we found exact equality (we won't!)
            if (a * a == 2 * b * b) {
                System.out.println("FOUND EXACT: " + a + "/" + b);
                return;
            }
        }
        
        System.out.println("\nNo exact rational representation found!");
        System.out.println("This is consistent with √2 being irrational.");
    }
    
    /**
     * Demonstrates the contradiction in the √2 proof.
     * If √2 = a/b in lowest terms, then both a and b must be even.
     */
    public static void verifyLowestTermsContradiction() {
        System.out.println("\n--- Demonstrating the Contradiction ---");
        
        // Suppose √2 = a/b with small values
        // If 2b² = a², what happens?
        
        for (int b = 1; b <= 20; b++) {
            int aSquared = 2 * b * b;
            int a = (int) Math.sqrt(aSquared);
            
            // Check if a² = 2b² exactly
            if (a * a == aSquared) {
                int gcd = gcd(a, b);
                System.out.printf("If √2 = %d/%d: gcd(%d,%d) = %d", a, b, a, b, gcd);
                
                if (gcd > 1) {
                    System.out.println(" → NOT in lowest terms!");
                } else {
                    System.out.println(" → In lowest terms");
                }
            }
        }
        
        System.out.println("\nNotice: Any 'solution' is never in lowest terms!");
    }
    
    /**
     * Computes the greatest common divisor using Euclidean algorithm.
     */
    public static int gcd(int a, int b) {
        while (b != 0) {
            int temp = b;
            b = a % b;
            a = temp;
        }
        return a;
    }
}

Exploring Euclid's Prime Construction

import java.util.*;

/**
 * Explores Euclid's construction for finding new primes.
 * Given a list of primes, constructs N = p₁·p₂·...·pₙ + 1
 * and finds its prime factors.
 */
public class EuclidsConstruction {
    
    public static void main(String[] args) {
        // Start with first few primes
        List<Long> primes = new ArrayList<>();
        primes.add(2L);
        
        System.out.println("Exploring Euclid's Construction:\n");
        
        for (int i = 0; i < 8; i++) {
            analyzeConstruction(primes);
            
            // Add next prime for next iteration
            long nextPrime = nextPrimeAfter(primes.get(primes.size() - 1));
            primes.add(nextPrime);
        }
    }
    
    /**
     * Given a list of primes, computes N = product + 1
     * and analyzes its factors.
     */
    public static void analyzeConstruction(List<Long> primes) {
        // Compute product of all primes in list
        long product = 1;
        for (long p : primes) {
            product *= p;
        }
        
        long N = product + 1;
        
        System.out.print("Primes: " + primes);
        System.out.println(", Product = " + product + ", N = " + N);
        
        // Find prime factors of N
        List<Long> factors = primeFactors(N);
        System.out.println("  Prime factors of N: " + factors);
        
        // Check if any factor is in original list
        boolean foundInList = false;
        for (long factor : factors) {
            if (primes.contains(factor)) {
                foundInList = true;
                break;
            }
        }
        
        if (foundInList) {
            System.out.println("  ERROR: Factor found in original list!");
        } else {
            System.out.println("  ✓ All factors are NEW primes (not in list)");
        }
        
        if (isPrime(N)) {
            System.out.println("  Note: N itself is prime!");
        } else {
            System.out.println("  Note: N is composite");
        }
        
        System.out.println();
    }
    
    /**
     * Returns list of prime factors of n.
     */
    public static List<Long> primeFactors(long n) {
        List<Long> factors = new ArrayList<>();
        
        for (long p = 2; p * p <= n; p++) {
            while (n % p == 0) {
                if (!factors.contains(p)) {
                    factors.add(p);
                }
                n /= p;
            }
        }
        
        if (n > 1) {
            factors.add(n);
        }
        
        return factors;
    }
    
    /**
     * Checks if n is prime.
     */
    public static boolean isPrime(long n) {
        if (n < 2) return false;
        if (n == 2) return true;
        if (n % 2 == 0) return false;
        
        for (long i = 3; i * i <= n; i += 2) {
            if (n % i == 0) return false;
        }
        return true;
    }
    
    /**
     * Returns the next prime after n.
     */
    public static long nextPrimeAfter(long n) {
        long candidate = n + 1;
        while (!isPrime(candidate)) {
            candidate++;
        }
        return candidate;
    }
}

💡 What the Code Demonstrates

  • The √2 explorer shows we can get arbitrarily close rational approximations but never exact
  • Any "solution" to 2b² = a² never has gcd(a,b) = 1
  • Euclid's construction always produces N with prime factors NOT in the original list
  • Sometimes N is prime, sometimes composite—but its factors are always "new"

Problem 1: Irrationality of √5

Problem Statement ▼

Prove: √5 is irrational.

Hint: Follow the same structure as the √2 proof. You'll need to show that if 5 | n², then 5 | n.

Solution ▼

Proof

Suppose, for contradiction, that √5 is rational. Then √5 = a/b where a, b ∈ ℤ, b ≠ 0, and gcd(a, b) = 1.

Squaring: 5 = a²/b², so 5b² = a².

Thus 5 | a². Since 5 is prime, 5 | a. Write a = 5k.

Substituting: 5b² = 25k², so b² = 5k².

Thus 5 | b², so 5 | b.

Both a and b are divisible by 5, contradicting gcd(a, b) = 1.

Therefore, √5 is irrational. ∎

Problem 2: log₂(3) is Irrational

Problem Statement ▼

Prove: log₂(3) is irrational.

Hint: If log₂(3) = a/b, then 2^(a/b) = 3, so 2^a = 3^b. Consider the prime factorizations.

Solution ▼

Proof

Suppose, for contradiction, that log₂(3) is rational.

Then log₂(3) = a/b for some positive integers a and b.

This means 2^(a/b) = 3, so (2^(a/b))^b = 3^b, giving 2^a = 3^b.

But 2^a is a power of 2 (even for a ≥ 1), and 3^b is a power of 3 (odd).

By the Fundamental Theorem of Arithmetic, 2^a and 3^b have different prime factorizations, so they cannot be equal.

This is a contradiction!

Therefore, log₂(3) is irrational. ∎

Problem 3: No Three Consecutive Integers Multiply to a Power of 2

Problem Statement ▼

Prove: There are no three consecutive positive integers whose product is a power of 2.

Hint: Consider what happens with odd numbers in any three consecutive integers.

Solution ▼

Proof

Suppose, for contradiction, that n(n+1)(n+2) = 2^k for some positive integers n and k.

Among any three consecutive integers, at least one is odd (in fact, exactly one or two are odd).

If n is odd, then n is an odd factor of the product.

If n is even, then n+1 is odd, so n+1 is an odd factor.

In either case, the product n(n+1)(n+2) has an odd factor greater than 1.

But 2^k has no odd factors greater than 1.

This is a contradiction!

Therefore, no three consecutive positive integers have a product that is a power of 2. ∎
