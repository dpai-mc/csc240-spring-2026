üìä Finite Sets and Cardinality

Comparing Finite Sets

For finite sets, comparing cardinalities is straightforward:

üîó Bijections and Cardinality

Here's a powerful idea: Two sets have the same cardinality if we can create a perfect pairing between their elements.

üìñ Key Concept: Cardinality via Bijections

Two sets A and B have the same cardinality (written |A| = |B|) if and only if there exists a bijective function f: A ‚Üí B.

|A| = |B| ‚ü∫ ‚àÉ bijection f: A ‚Üí B

Bijection means: One-to-one (injective) AND onto (surjective)

Why Bijections Matter

üéØ The Power of Bijections

Bijections let us compare set sizes without counting! This is crucial for infinite sets where we can't count directly.

  • Each element in A maps to exactly one element in B
  • Each element in B is mapped to by exactly one element in A
  • Perfect one-to-one correspondence
  • Works for both finite AND infinite sets!

üí° Example: Simple Bijection

A = {1, 2, 3} and B = {a, b, c}

Define f: A ‚Üí B as:

  • f(1) = a
  • f(2) = b
  • f(3) = c

This is a bijection! Each element in A pairs with exactly one element in B, and vice versa.

Conclusion: |A| = |B| = 3

Problem: Prove that the set of all even integers has the same cardinality as the set of all odd integers.

Hint: Construct an explicit bijection between the two sets.

Click to see solution

Solution:

Let E = {even integers} and O = {odd integers}

Define f: E ‚Üí O by f(n) = n + 1

  • f is well-defined: if n is even, n+1 is odd
  • f is injective: if f(m) = f(n), then m+1 = n+1, so m = n
  • f is surjective: for any odd k, k-1 is even and f(k-1) = k

Therefore f is a bijection, so |E| = |O|. ‚àé
