Problem 1: Cardinality and Bijections
2 points

a) (1 point) Prove that the set of even natural numbers E = {2, 4, 6, 8, ...} has the same cardinality as ‚Ñï (the set of natural numbers ‚Ñï = {1, 2, 3, 4, ...}) by constructing an explicit bijection.

b) (1 point) Explain why this result is counterintuitive. What does it tell us about infinite sets?

üí° Hint: For constructing a bijection

A bijection needs to be one-to-one (injective) and onto (surjective). Think about a simple arithmetic operation that maps each natural number to a unique even natural number, and vice-versa.

  • Define a function f: ‚Ñï ‚Üí E.
  • Prove f is injective: if f(n‚ÇÅ) = f(n‚ÇÇ), then n‚ÇÅ = n‚ÇÇ.
  • Prove f is surjective: for every e ‚àà E, there exists an n ‚àà ‚Ñï such that f(n) = e.

Example approach: Consider f(n) = 2n. Does this map every natural number to a unique even number?

üí° Hint: For the intuitive aspect

Consider the relationship between a set and its proper subset. How does our intuition about finite sets apply (or fail to apply) to infinite sets in terms of "size"?

Think about: For finite sets, a proper subset is always smaller. Does this hold for infinite sets?

Key insight: The even numbers are a proper subset of natural numbers (E ‚äÇ ‚Ñï), yet they have the same cardinality!

Problem 2: Countability
2 points

Determine whether each set is finite, countably infinite, or uncountably infinite. Justify your answers.

a) (0.5 points) The set of all integers between -1000 and 1000
b) (0.5 points) The set of all negative integers
c) (0.5 points) The set of all real numbers between 0 and 1 (exclusive)

üí° Hint: Definitions of countability

  • Finite: Can be counted, there's a last element. Has cardinality n for some natural number n.
  • Countably Infinite: Can be put into a one-to-one correspondence with the natural numbers ‚Ñï. Think "listable" - you can enumerate elements as a‚ÇÅ, a‚ÇÇ, a‚ÇÉ, ...
  • Uncountably Infinite: Cannot be put into a one-to-one correspondence with ‚Ñï. Think "too many to list."

Key facts to remember:

  • ‚Ñï, ‚Ñ§, and ‚Ñö are all countably infinite
  • ‚Ñù and any interval of real numbers are uncountably infinite
  • Recall Cantor's diagonal argument for real numbers
  • A finite subset of any set is finite
Problem 3: Logical Propositions
2 points

For each statement, determine if it is a proposition. If it is, state its truth value (TRUE or FALSE). If it is not a proposition, explain why.

a) (0.4 points) "2 + 3 = 5"
b) (0.4 points) "Is it raining?"
c) (0.4 points) "This statement is false"
d) (0.4 points) "All prime numbers greater than 2 are odd"
e) (0.4 points) "x¬≤ = 9"

üí° Hint: Definition of a proposition

A proposition must be a declarative statement that is either true or false, but not both.

NOT propositions:

  • Questions are not propositions (they ask, not state).
  • Commands are not propositions (they direct action).
  • Statements with variables are generally not propositions unless the variable is quantified or has a specific value.
  • Paradoxical statements (like "This statement is false") are not propositions because they can't consistently be true or false.
  • Opinions without objective truth values are not propositions.

For each statement, ask:

  1. Is it a declarative statement?
  2. Does it have a definite truth value?
  3. Can it be definitely true or definitely false (not both)?

üí° Hint: Truth table construction

Step-by-step process:

  1. Step 1: Identify all distinct propositional variables. This tells you the number of rows (2‚Åø).
  2. Step 2: Create columns for each variable and list all possible truth value combinations.
  3. Step 3: Create intermediate columns for sub-expressions, following operator precedence (¬¨ highest, then ‚àß, ‚à®, ‚Üí, ‚Üî lowest).
  4. Step 4: The last column is for the complete expression.
  5. Step 5: Classify the result:
    • All T's = Tautology
    • All F's = Contradiction
    • Mixed = Contingency

Remember:

  • p ‚àß q is true only when BOTH are true
  • p ‚à® q is false only when BOTH are false
  • p ‚Üí q is false only when p is TRUE and q is FALSE
  • ¬¨ flips the truth value

p | q | ¬¨p | ¬¨p ‚àß q
--|---|----|---------

T | F | F  | F

üí° Hint: Verifying logical equivalence

Two expressions are logically equivalent if and only if their truth tables are identical.

Method:

  1. Construct a single truth table for each pair
  2. Include columns for both expressions
  3. Compare their final columns row by row
  4. If all rows match ‚Üí expressions are equivalent
  5. If even one row differs ‚Üí not equivalent
