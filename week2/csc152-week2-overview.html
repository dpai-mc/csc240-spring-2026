Week 2 Overview

Week Overview

Week 2 builds upon the foundational server concepts from Week 1 by diving deep into server hardware architecture and components. Students will learn about the critical hardware elements that make servers reliable and performant, including processors, memory, storage, network interfaces, and power/cooling systems. This week emphasizes understanding hardware specifications, monitoring server resources, and making informed decisions about hardware selection and configuration. Students will also explore data center environmental considerations that ensure optimal server operation.

Learning Objectives

By the end of Week 2, students will be able to:

  1. Identify and explain the function of major server hardware components (CPU, RAM, storage, NICs)
  2. Install and configure server hardware components in virtual environments
  3. Describe power and cooling requirements for servers and data centers
  4. Explain environmental controls necessary for reliable server operation
  5. Use hardware monitoring tools to assess server resource utilization
  6. Compare and contrast different storage technologies (HDD, SSD, NVMe)
  7. Evaluate hardware specifications for different server workloads
  8. Calculate power consumption and cooling requirements for server deployments
CSLO Alignment CSLO 1 - Manage and maintain servers, including operating system (OS) configuration, access control and virtualization
