📌 Scope Note — First Exposure Only

This module is a conceptual introduction. You will learn what a generating function is and how to read its coefficients. Algebraic manipulation of generating functions — multiplying, dividing, shifting, and using them to solve recurrence relations — is covered fully in Week 11. Your goal right now is recognition and coefficient extraction, not manipulation.

1 · Motivation — One Object, Infinitely Many Answers

In Modules 8.1–8.3 you built a powerful toolkit: the Binomial Theorem, four key binomial coefficient identities, multinomial coefficients, and the stars-and-bars method. Each tool answers a specific type of counting question. But what if you could package all the answers to a family of related counting questions — one answer for each size n = 0, 1, 2, 3, … — into a single algebraic expression?

That is precisely what a generating function does.

The Filing-Cabinet Analogy

Imagine a numbered filing cabinet. Each drawer is labeled with a non-negative integer n, and inside drawer n sits the answer to your counting question for size n. A generating function is the algebraic encoding of that entire cabinet — you extract the answer for any particular n simply by reading the coefficient of xn.

💡 The Key Idea

The filing cabinet above holds the binomial coefficients C(7,0), C(7,1), …, C(7,7). The single expression (1 + x)7 encodes all eight values at once. To retrieve C(7,3) = 35, you simply read the coefficient of x3. That expression is the generating function for this sequence.

Why Should a Computer Scientist Care?

Generating functions appear throughout computer science in roles you will encounter throughout this course and beyond:

  • Solving recurrence relations — the Fibonacci sequence, merge-sort complexity, and many algorithm analyses reduce to algebraic operations on generating functions (Week 11).
  • Counting with constraints — generating functions make it easy to encode restrictions like "at most 3 of this type" or "an even number of that type."
  • Probability distributions — the binomial distribution you will see in Week 9 is directly related to the generating function (p + q)n.
  • Algorithm analysis — the generating function perspective connects counting inputs to asymptotic complexity (Module 8.5).

2 · Formal Definition

Definition — Ordinary Generating Function

Let {a0, a1, a2, …} be a sequence of real numbers. The ordinary generating function of this sequence is the formal power series:

G(x)  =  a0 + a1x + a2x2 + a3x3 + ···  =  ∑n=0∞  an xn

The coefficient an is the answer to the counting question for size n.

What "Formal" Means

The word formal is important. In a formal power series, x is not a number we substitute — it is a placeholder or "slot marker." We are not asking whether the series converges; we are using the algebraic structure of powers of x to keep the terms of the sequence organized and separated.

Think of it this way: the exponent on x is just a label telling you which term of the sequence you are looking at. x5 means "this is the answer for n = 5."

✅ The Reading Rule

To find the answer an from a generating function G(x): identify the coefficient of xn in G(x). That coefficient is an.

3 · Two Fundamental Examples

Example 1 — The Binomial Theorem as a Generating Function

You already know the Binomial Theorem from Module 8.1. Viewed through the lens of generating functions, it says something beautiful:

The Binomial GF

The generating function G(x) = (1 + x)n encodes the sequence of binomial coefficients C(n,0), C(n,1), C(n,2), …, C(n,n):

(1 + x)n  =  C(n,0)·x⁰ + C(n,1)·x¹ + C(n,2)·x² + ··· + C(n,n)·xⁿ

Reading the GF: The coefficient of xk in (1+x)n is C(n,k) — the number of k-element subsets of an n-element set.

Question: Using the generating function G(x) = (1+x)6, find the number of 0-element, 2-element, and 4-element subsets of a 6-element set.

  1. 1 The coefficient of xk in (1+x)6 is C(6,k).
  2. 2 For k=0: coefficient of x0 is C(6,0) = 1. (There is exactly 1 empty subset.)
  3. 3 For k=2: coefficient of x2 is C(6,2) = 15. (There are 15 two-element subsets.)
  4. 4 For k=4: coefficient of x4 is C(6,4) = C(6,2) = 15. (Symmetry identity confirms this.)
✅ Connection to Module 8.2
The symmetry C(6,4) = C(6,2) is visible directly in the generating function: the coefficients of x4 and x2 are equal. This is the symmetry identity from Module 8.2 expressed as a property of the power series.

Example 2 — The Geometric Series

The second fundamental example encodes a much simpler sequence — the constant sequence 1, 1, 1, 1, …

The Geometric Series GF

The generating function for the constant sequence an = 1 for all n ≥ 0 is:

G(x)  =  11 − x  =  1 + x + x2 + x3 + x4 + ···  =  ∑n=0∞  xn

Plain-language meaning: "There is exactly 1 way to choose exactly n identical items from an unlimited supply" — for every value of n. The coefficient of every power of x is 1.

Let S = 1 + x + x2 + x3 + ··· (treating x as a formal variable with |x| < 1 for convergence purposes).

  1. 1 Multiply both sides by x: xS = x + x2 + x3 + ···
  2. 2 Subtract: S − xS = 1 (all other terms cancel).
  3. 3 Factor: S(1 − x) = 1.
  4. 4 Divide: S = 1/(1−x). ✓

In the formal power series setting we simply define 1/(1−x) to be the series ∑ xn. Convergence is not required for the algebraic manipulations we use in Week 11.

Side-by-Side Comparison

Generating Function G(x) Sequence {an} Counting Meaning Coefficient of xk
(1 + x)n C(n,0), C(n,1), …, C(n,n), 0, 0, … # of k-element subsets of an n-element set C(n, k)
1/(1 − x) 1, 1, 1, 1, … # of ways to choose k identical items from unlimited supply 1 (for all k ≥ 0)

4 · Coefficient Extraction Practice

The core skill for this module is reading a generating function and extracting the coefficient that answers your counting question. Work through the three annotated examples below, then check your understanding in Section 6.

Identify the generating function: G(x) = (1+x)7. This encodes the binomial coefficients C(7,k).

  1. 1 We want the coefficient of x3, so we need k = 3.
  2. 2 The coefficient of xk in (1+x)n is C(n,k).
  3. 3 Compute: C(7,3) = 7!/(3!·4!) = (7·6·5)/(3·2·1) = 35.
[x³] (1+x)⁷  =  C(7,3)  =  35

Counting interpretation: There are 35 three-element subsets of a 7-element set.

Identify the generating function: G(x) = 1/(1−x) = ∑ xn. Every coefficient in this series is 1.

  1. 1 Expand: G(x) = 1 + x + x2 + x3 + x4 + ···
  2. 2 The coefficient of x4 is the number in front of x4, which is 1.
[x⁴]  11−x  =  1

Counting interpretation: There is exactly 1 way to select 4 identical objects from an unlimited supply (you simply take 4 of them — there is no choice to make).

Identify the pattern: The sequence is an = 2n.

  1. 1 Write the generating function: G(x) = ∑ 2n xn.
  2. 2 Rewrite each term: 2n xn = (2x)n.
  3. 3 Recognize a geometric series with ratio 2x: ∑ (2x)n = 1/(1−2x).
G(x)  =  11 − 2x  =  1 + 2x + 4x² + 8x³ + 16x⁴ + ···
⚠️ Scope Reminder
You are not expected to derive closed forms like 1/(1−2x) from scratch yet. The goal here is to recognize the pattern and verify that the coefficient of xn in 1/(1−2x) is indeed 2n. Algebraic manipulation of generating functions is Week 11 content.

5 · Looking Ahead

📅 Week 9 — Discrete Probability

In Week 9 you will study the Binomial Distribution, which models the probability of exactly k successes in n independent trials. The formula involves C(n,k) — the same coefficient you just read from the generating function (1+x)n. Counting becomes the numerator of probability.

The generating function of a probability distribution is called a probability generating function. It is a direct descendant of the ordinary generating functions introduced here.

📅 Week 11 — Recurrence Relations

In Week 11 you will learn to manipulate generating functions — adding, multiplying, and shifting them — to solve recurrence relations. For example, the Fibonacci sequence F(n) = F(n−1) + F(n−2) has the generating function:

G(x)  =  x1 − x − x²

Reading the coefficient of xn in that expression gives the n-th Fibonacci number in closed form. You now have all the conceptual vocabulary needed to understand that technique when you encounter it in Week 11.
