Vandermonde Systems, Cramer's Rule, and First-Order Recurrences

Supplementary reading · CSC 240 Discrete Structures · Week 14

📌 Purpose of this reading

This page fills in the linear-algebra machinery behind solving second-order recurrence relations. When a recurrence has two distinct characteristic roots, finding the closed-form solution requires solving a 2×2 linear system. That system has a Vandermonde coefficient matrix, and Cramer's Rule gives a clean determinant-based formula for its solution. The first-order case (one root) is also covered because it is dramatically simpler.

1 · Background: Second-Order Recurrences

1.1 What Is a Second-Order Recurrence?

Second-Order Linear Homogeneous Recurrence. A rule of the form

T(n) = c₁ · T(n−1) + c₂ · T(n−2), for n ≥ 2

where c₁ and c₂ are constants (c₂ ≠ 0). Second-order means the current term depends on the two immediately preceding terms. Homogeneous means no additional function of n is added on the right side.

Two initial values — T(0) and T(1) — are needed to determine a unique sequence. Given those values, every subsequent term is uniquely defined by the recurrence.

Quick example — building a sequence from a recurrence

Consider T(n) = 5T(n−1) − 6T(n−2) with T(0) = 2 and T(1) = 5.

  • T(2) = 5·5 − 6·2 = 25 − 12 = 13
  • T(3) = 5·13 − 6·5 = 65 − 30 = 35
  • T(4) = 5·35 − 6·13 = 175 − 78 = 97

The recurrence computes each term, but it does not tell us a closed-form formula like T(n) = f(n). Finding that formula is what the rest of this reading is about.

1.2 The Characteristic Equation

The standard technique for finding a closed-form solution is to look for solutions of the form T(n) = rⁿ. Substituting into T(n) = c₁T(n−1) + c₂T(n−2):

rⁿ = c₁ · rⁿ⁻¹ + c₂ · rⁿ⁻²

Divide both sides by rⁿ⁻² (valid when r ≠ 0):

r² = c₁r + c₂, which rearranges to r² − c₁r − c₂ = 0

Characteristic Equation. The quadratic r² − c₁r − c₂ = 0 associated with T(n) = c₁T(n−1) + c₂T(n−2). Its roots r₁ and r₂ are called the characteristic roots of the recurrence.

The characteristic roots tell you which exponential functions are solutions of the recurrence. In the example T(n) = 5T(n−1) − 6T(n−2), the characteristic equation is r² − 5r + 6 = 0, which factors as (r − 2)(r − 3) = 0, giving roots r₁ = 2 and r₂ = 3.

1.3 The General Solution (Two Distinct Roots)

When the two characteristic roots are distinct (r₁ ≠ r₂), both rⁿ₁ and rⁿ₂ satisfy the recurrence, and so does any linear combination of them. It can be shown that the most general solution is:

T(n) = A₁ · r₁ⁿ + A₂ · r₂ⁿ

The constants A₁ and A₂ are not yet determined — they depend on the specific initial values T(0) and T(1). Substituting n = 0 and n = 1 into this formula produces two equations in two unknowns, which is the 2×2 linear system that the rest of this reading shows how to solve.

⚠️ Distinct roots only

The formula T(n) = A₁r₁ⁿ + A₂r₂ⁿ applies only when the two characteristic roots are different. When r₁ = r₂ (a repeated root), the solution form changes to T(n) = (A₁ + A₂n)rⁿ. That case is not covered in this reading.

2.2 The Vandermonde Pattern

2×2 Vandermonde Matrix. A matrix of the form

Each column corresponds to one characteristic root (column 1 → r₁, column 2 → r₂). Each row corresponds to a power of those roots: row 1 uses the 0th power (so both entries are 1), and row 2 uses the 1st power (entries are r₁ and r₂). In the general n×n Vandermonde matrix this pattern continues to higher powers, but in the 2×2 case only powers 0 and 1 appear.

💡 Key Fact

The Vandermonde determinant equals r₂ − r₁. This is nonzero precisely when r₁ ≠ r₂. So whenever the characteristic roots are distinct, the linear system has exactly one solution — one pair of constants A₁ and A₂ that satisfies both initial conditions.

If r₁ = r₂ (repeated root), the determinant is 0, the Vandermonde approach fails, and the different solution form T(n) = (A₁ + A₂n)rⁿ must be used instead.

3 · Cramer's Rule

3.1 What Cramer's Rule Says

Cramer's Rule (2×2 case). Given a 2×2 system Ax = b where D = det(A) ≠ 0:

x₁ = D₁ / D x₂ = D₂ / D

3.3 Worked Example

📝 Example

A recurrence has characteristic roots r₁ = 2, r₂ = 3 and initial values T(0) = 3, T(1) = 7. Find the closed-form solution T(n).

Show full solution

Step 1 — Write the solution form. Since r₁ ≠ r₂:

T(n) = A₁ · 2ⁿ + A₂ · 3ⁿ

Step 2 — Set up the linear system.

n = 0: A₁ + A₂ = 3 n = 1: 2A₁ + 3A₂ = 7

Step 3 — Compute the three determinants.

D = r₂ − r₁ = 3 − 2 = 1 D₁ = T(0)r₂ − T(1) = 3·3 − 7 = 9 − 7 = 2 D₂ = T(1) − r₁T(0) = 7 − 2·3 = 7 − 6 = 1

Step 4 — Apply Cramer's Rule.

A₁ = D₁ / D = 2 / 1 = 2 A₂ = D₂ / D = 1 / 1 = 1

Step 5 — Write the closed-form solution.

T(n) = 2 · 2ⁿ + 1 · 3ⁿ

Verification:

T(0) = 2·1 + 1·1 = 3 ✓ T(1) = 2·2 + 1·3 = 4 + 3 = 7 ✓ T(2) = 2·4 + 1·9 = 8 + 9 = 17 T(3) = 2·8 + 1·27 = 16 + 27 = 43

4 · The First-Order Case

4.1 Why Only One Initial Condition Is Needed

A first-order homogeneous recurrence has the form T(n) = c · T(n−1). Its characteristic equation is simply r = c, giving one root. The solution form with one unknown is:

T(n) = A₁ · rⁿ

Now substitute n = 0. Since r⁰ = 1 for any r:

T(0) = A₁ · 1 → A₁ = T(0)

There is no system to solve. The one constant equals the initial value directly.

✅ Closed-form result

For any first-order homogeneous recurrence T(n) = c · T(n−1) with initial value T(0):

T(n) = T(0) · cⁿ

This is a geometric sequence with first term T(0) and common ratio c. No Vandermonde matrix, no Cramer's Rule needed.

4.2 Example

T(n) = 3 · T(n−1) with T(0) = 5. Then T(n) = 5 · 3ⁿ.

✅ Summary
  • A second-order recurrence T(n) = c₁T(n−1) + c₂T(n−2) has characteristic equation r² − c₁r − c₂ = 0.
  • With two distinct roots r₁ and r₂, the solution form is T(n) = A₁r₁ⁿ + A₂r₂ⁿ. Plugging in n = 0 and n = 1 produces a 2×2 Vandermonde system.
  • The 2×2 Vandermonde determinant is r₂ − r₁. When r₁ ≠ r₂ this is nonzero, guaranteeing a unique solution via Cramer's Rule: A₁ = (T(0)r₂ − T(1))/(r₂ − r₁) and A₂ = (T(1) − r₁T(0))/(r₂ − r₁).
  • The first-order case T(n) = c · T(n−1) has exactly one root and one unknown: A₁ = T(0) directly, giving T(n) = T(0) · cⁿ with no matrix system needed.
